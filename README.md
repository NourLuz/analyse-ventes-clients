# 📊 Analyse des ventes et segmentation clients (RFM)

## 🧠 Contexte
Dans un environnement commercial, il est essentiel de comprendre le comportement des clients afin d’optimiser les ventes et la fidélisation.

## 🎯 Objectif
Analyser les données de ventes pour :
- Identifier les clients les plus rentables
- Détecter les clients à risque
- Suivre l’évolution du chiffre d’affaires

## 🛠️ Outils utilisés
- Python (Pandas, NumPy, Matplotlib)
- Power BI
- Excel

## 📁 Structure du projet
- data/ : données brutes
- python/ : traitement et analyse
- dashboard/ : visualisation Power BI

## 📈 Analyses réalisées
- Nettoyage et préparation des données
- Calcul du chiffre d’affaires
- Analyse temporelle des ventes
- Segmentation RFM (Récence, Fréquence, Montant)

## 📊 Dashboard
Un tableau de bord interactif a été réalisé permettant :
- Visualisation des ventes mensuelles
- Analyse des segments clients (VIP, Fidèle, À risque, Perdu)
- Aide à la prise de décision

## 🚀 Résultats clés
- Identification des périodes de forte et faible activité
- Mise en évidence des clients à forte valeur
- Détection des clients inactifs

## 💡 Compétences démontrées
- Manipulation de données (Pandas)
- Analyse métier
- Data visualisation
- Utilisation d’un outil BI
